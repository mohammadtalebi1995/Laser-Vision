# Industrial Machine Vision – Defect Detection

A laser scanner is a device that analyzes objects in the real world or the surrounding environment and collects data digitally based on their possible shape and appearance. A laser scanner is one of the latest digitizer systems for 3D measurement, which is measured using laser light. This system is flexible according to the user's needs, and the parameters that can determine the flexibility of this scanner are:
 The surface on which the control is performed
 The level of accuracy required by the customer.
 The environment in which this technology is used.

The way a laser scanner works is that by using the image of the laser beam shining on the surface, an image similar to the one below is obtained and the three-dimensional dimensions are examined and inserted using software.

Laser scanner applications:
The following are among the applications of a laser scanner:
 Control of deposits in welding points
 Extraction of 3D files for various types of modeling
 Calculation of width and height dimensions
 Precise measurements
 Examination of protrusions and depressions in parts
